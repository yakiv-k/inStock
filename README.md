### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/yakiv-k/inStock.git
```

Go to the project directory

```bash
  cd instock-deployed
```

Install dependencies

```bash
  npm i
```
To start this project run

```bash
  npm start
```
